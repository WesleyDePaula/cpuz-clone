## Como executar
Instalação do compilador GCC:
- Instale o compilador GCC -> https://sourceforge.net/projects/mingw-w64/
- Localize o local de instalação (geralmente está no diretório ``C:\MinGW\bin``) e adicione nas variáveis de ambiente
- Verifique se foi instalado corretamente executando o comando ``gcc --version`` no terminal

Execução do programa:
- Clone o projeto
- Execute o script ./compile.sh que está na pasta raiz do projeto
- Irá criar um arquivo executável (.exe) do programa na raiz do projeto
